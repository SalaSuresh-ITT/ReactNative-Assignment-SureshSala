# ReactNative-Assignment-SureshSala
React Native assignment from MernPlus

## Steps to Run the application in mobile device:
Before cloning the project in to local, make sure React Native development environment setuped in computer.
Please refer the below link for setup.
[https://reactnative.dev/docs/environment-setup](https://reactnative.dev/docs/environment-setup)

Clone the project and open the project folder in Visual Studio Code editor.
Please refer the below link to download Visual Studio Code.
[https://code.visualstudio.com/](https://code.visualstudio.com/)

Open command prompt in project location and run the below command to install/generate library dependency files.
```bash
npm install
```

Once the installation is done, use below command to run the project in Android mobile.
```bash
react-native run-android
```

Before the running the applicaiton in iOS Device. Need to install ios dependency/pod files, to install those use below commands in project location.
```bash
cd ios
pod install
```

Once pods/ios dependency files are installed, open the ios project in XCode and run project.
To open the ios project in XCode, double click on Assignment_Proj.xcworkspace file, under below link.
```bash
<PROJECT LOCATION>/ReactNative-Assignment-SureshSala/ios/Assignment_Proj.xcworkspace
```
